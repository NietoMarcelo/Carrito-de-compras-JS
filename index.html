Descripción del simulador

Este simulador permite gestionar un carrito de compras de forma interactiva mediante un menú. El usuario puede agregar productos, ver el carrito completo, editar productos existentes, eliminar productos específicos o vaciar todo el carrito. Todas las interacciones se realizan mediante cuadros de diálogo (Prompt, Alert, Confirm).

Instrucciones: El simulador comenzará automáticamente al cargar la página. Seguí las indicaciones en los cuadros de diálogo y seleccioná la opción del menú ingresando el número correspondiente (1 al 6).
